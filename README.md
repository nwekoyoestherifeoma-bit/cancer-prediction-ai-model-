# Cancer Prediction AI Model

I built an AI model that can predict whether a tumour 
is cancerous or not.

## Dataset
- Source: Wisconsin Breast Cancer Dataset
- 569 real patients
- 30 medical measurements per patient

It was trained on data from 569 real patients and 
achieved 97.37% accuracy.

## Tools Used
- Python
- Scikit-learn
- Pandas
- Numpy 
- Jupyter Notebook

## How The Model Works
1. Loaded real medical data from 569 patients
2. Split data into 80% training and 20% testing
3. Trained a Random Forest Classifier
4. Tested accuracy on 114 unseen patients
5. Achieved 97.37% accuracy
